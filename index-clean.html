FAQ

Questions frequentes

OmnIA remplace-t-elle ChatGPT ou Claude ? +
Non. OmnIA utilise ces modeles — elle ne les remplace pas. C'est la couche de controle, de gouvernance et d'orchestration qui se place au-dessus. Vos equipes continuent de converser avec Claude, GPT, Mistral — mais via un hub securise et audite.
Ou sont hebergees mes donnees ? +
Sur votre infrastructure. OmnIA se deploie on-premise via Docker ou sur votre cloud prive. Aucune donnee metier ne transite en dehors de votre perimetre. Les seuls flux sortants sont les appels aux APIs LLM que vous avez autorises.
Que se passe-t-il si j'arrete le contrat ? +
Le logiciel continue de tourner. Vous conservez le code source, toutes vos donnees, et une documentation complete. Aucune clause de non-concurrence, aucun frais de sortie. Zero lock-in.
Puis-je changer de fournisseur LLM ? +
Oui, a tout moment. LiteLLM agit en proxy uniforme. Si vous passez d'Azure OpenAI a Mistral, ou ajoutez un modele on-premise, c'est une reconfiguration — pas un projet. Aucune ligne de code OmnIA ne change.
Faut-il savoir coder pour utiliser OmnIA ? +
Non. OmnIA est concue pour les directions metier : DSI, conformite, RH, finance, juridique. Creer un agent prend quelques minutes via un formulaire guide. L'administration se fait via une interface graphique complete.
Quel est le perimetre de l'ecosysteme Golden Age ? +
Golden Age est l'ecosysteme complet qui englobe : PulseBoard (infra & control), PearlFlow (pipelines agentiques), MCP Universe (interoperabilite), Migration Factory (Migratorflow + MigratorBI), et les applications metiers (OmnIA Portal, Harry Helper, SlideWise). Chaque module peut etre deploye independamment.
